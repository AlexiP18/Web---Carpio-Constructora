### Caso 1: Agregar Nuevo Proyecto

#### Con Decap CMS:
```
1. Ir a /admin/
2. Collections → Proyectos → Nuevo
3. Llenar formulario (sin saber cómo se ve)
4. Agregar imágenes (sin preview)
5. Guardar
6. Esperar build (5 min)
7. Ver resultado
8. "¡No! Ese no es el orden que quería"
```

#### Con Tina CMS:
```

4. Arrastrar y soltar imágenes (con preview)
5. Ajustar todo mientras ves el resultado
6. Guardar (1 click)
Tiempo total: 5 minutos
```

**Ahorro: 75% del tiempo**

## 🔥 KILLER FEATURES DE TINA

### 1. Block-based Content

**¿Qué es?**
Permite crear contenido con bloques drag & drop.

**Ejemplo:**
```typescript
// tina/config.ts
{
  type: 'object',
  name: 'blocks',
  list: true,
  templates: [
    {
      name: 'hero',
      label: 'Hero Section',
      fields: [...]
    },
    {
      name: 'features',
      label: 'Features Grid',
      fields: [...]
    },
    {
      name: 'cta',
      label: 'Call to Action',
      fields: [...]
    },
  ]
}
```

**Resultado:**
El cliente puede arrastrar y soltar secciones completas.
